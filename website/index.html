FEATURES

AdsProphet is a novel mobile ads library that is based on the observation that when using apps, user's a lot of time has been wasted on waiting for the app to load remote data and such ''freezing time'' has not been fully taken advantage of. AdsProphet can find these valuable windows and displays full-screen ads (or so called interstitial ads), which attracts user attention 25 times higher than what banner ads can achieve. By using AdsProphet, developers' revenue can be expected to be increased significantly.

Novel Prediction Algorithm

AdsProphet adopts hierarchical user-perceived delay prediction algorithm that can effeiciently and accurately estimate user's waiting time for each networking operation.

More Ads Revenue

AdsProphet can automatically find opportunities to play full-screen ad, which has a 25 times higher Click-Through Rate (CTR) than banner ad, but is rarely used by developers because it competes app contents.

Complementary to Existing Libraries

AdsProphet is compelmentary to other ads library. Developers are free to use AdsProphet as well as other libraries to play banner ads, in order to maximize their revenue.

Sponsored by NSF I-Corps Program

AdsProphet is a start-up from Northwestern University's EECS Department. AdsProphet is sponsored by NSF I-Corps Program, which teaches entrepreneurship to students and faculty.
